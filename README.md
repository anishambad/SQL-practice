# SQL Practice Repository

This repository contains my structured SQL practice, organized by files.  
Each file represents a set of exercises focused on specific SQL concepts.

## Structure

The repository is organized as:

- File_01_Exercises.sql
- File_02_Exercises.sql
- File_03_Exercises.sql
...

Each file includes:
- Multiple SQL queries
- Problem statements (in comments)
- Concepts covered (documented inside the file)
- Clean and readable query formatting

## How to Read

Each SQL file follows this format:

1. Header Section  
   - File name / Exercise number  
   - Topics covered  

2. Queries  
   - Each query is labeled (Q1, Q2, ...)  
   - Problem statement included as comments  

3. Notes (inside SQL file)  
   - Key learnings  
   - Best practices  
   - Optimizations (if any)

## Topics Covered

- SELECT statements
- Filtering (WHERE)
- DISTINCT
- Aliases
- Expressions and calculations
- Joins
- Aggregations
- Subqueries
- Window functions

## Approach

- Practicing SQL through structured exercise
- Writing clean and optimized queries
- Documenting concepts directly within SQL files
- Improving problem-solving consistency

## Goal

To build strong SQL fundamentals and advance towards real-world data analysis and interview-level problem solving.
